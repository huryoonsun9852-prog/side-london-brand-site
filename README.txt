구조:
- index.html: 메인 스크롤 페이지 하나
- brochure.html: Brochure 카드 클릭 시 이동하는 별도 상세 페이지

- web.html: Web Header 상세 페이지 placeholder

반영:
1. index.html은 하나만 있습니다.
2. 메인 페이지의 Applications 섹션은 첨부 레퍼런스처럼 프로젝트 리스트/카드형 레이아웃입니다.
3. Brochure 카드를 누르면 brochure.html로 이동합니다.
4. 파란색 Download 텍스트는 없습니다.
